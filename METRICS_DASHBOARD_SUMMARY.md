# Grafana Dashboard для мониторинга приложения

## Обзор

Создан дашборд Grafana для визуализации метрик приложения в формате Prometheus. Дашборд включает:

- Статистику по альбомам, артикулам и файлам
- Мониторинг дискового пространства для различных точек монтирования
- Размер базы данных
- Время работы приложения
- Статистику HTTP-запросов (частота, коды состояния, время ответа)
- Количество активных подключений

## Файлы

- `grafana_dashboard.json` - JSON-описание дашборда для импорта в Grafana
- `GRAFANA_DASHBOARD_README.md` - подробная документация по дашборду

## Установка

1. Установите и настройте Grafana
2. Добавьте Prometheus в качестве источника данных
3. Импортируйте `grafana_dashboard.json` в Grafana через интерфейс
4. Назначьте источник данных Prometheus при импорте дашборда

## Мониторинг

Дашборд автоматически обновляется с интервалом, соответствующим настройкам Prometheus. Все метрики собираются из приложения через endpoint `/metrics`.
